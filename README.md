# Parent Teacher Management System

## Overview
Parent Teacher Management System (PTMS) is a web application that improves communication between parents and teachers. It allows parents to monitor student attendance, academic performance, announcements, and communicate directly with teachers through a messaging system.

## Features
- User Authentication and Authorization
- Parent Dashboard
- Student Attendance Tracking
- Academic Performance Monitoring
- Parent-Teacher Messaging System
- Announcements and Notifications

## Tech Stack
- Backend: Spring Boot
- Frontend: React.js
- Database: MySQL
- IDE: IntelliJ IDEA
- Build Tool: Maven
